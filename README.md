# Zaika-kart 🍔

**Zaika-kart** is a simple food website project built using HTML and CSS.  
This project is perfect for beginners to practice web development and features a responsive design.

## Features
- Responsive layout (works on desktop and mobile)  
- Attractive UI with images and buttons  
- Easy navigation between sections  

## Built With
- **HTML** – for website structure  
- **CSS** – for styling and responsive design  

### How It Was Built
- Used Flexbox & Grid for layout  
- CSS transitions and hover effects for better UI
